# ptp2021-7-2d-game
Игра "Magic Fight":
- 2D, вид сбоку
- Мультиплеерная онлайн-игра (от 2 игроков)
- Система магии: пользователь сначала комбинирует различные элементы,
а потом чертит из них заклинание на экране
- Цель игры: уничтожить противника(-ов)
